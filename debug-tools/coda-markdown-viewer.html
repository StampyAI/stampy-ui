Coda Raw Markdown Viewer

This tool shows you the raw markdown from the Coda API, exactly as it's received before any processing.

API Setup

To find your token: Open your wrangler.toml file and look for the line that says CODA_TOKEN = "your-token-here"

Code Path: The markdown is stored in response.items[0].values['Rich Text'] from the Coda API response.

Processing: In stampy-ui, it's extracted with extractText(values['Rich Text']) which handles arrays and removes code block markers (```)
